# Rails React Web App

> This project the user will be able to display random greetings that where save it in a DB table.


## Built With

- Ruby
- JavaScript
- Rails
- React
- PostgreSQL


## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Ruby
- Rails
- RubyGems

### Setup
**clone repository**
``
git clone <REPO-NAME>

``
### Install
**Install dependencies**
``
bundle install

``
### Usage
**Go to directory**
``
cd hello-rails-react

``
**Run Server**
``
rails s

``
